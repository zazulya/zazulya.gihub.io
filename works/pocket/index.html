Программа в состоянии нормально функционировать не только на компьютере у автора, а в рамках любой подходящей системы

Хватит таскать макулатуру на тренировку! На дворе 21 век, по чему бы не воспользоваться электронным дневником для ведения тренировок?
T Note — это приложение для учета занятий в спортзале. Главным принципом, при его создании, было «Меньше действий, выше эффективность». При правильном подходе, количество действий, для учета занятий, сводится к минимуму. Приложение автоматически заполняет каждую тренировку данными из прошлой, в результате, для большинства упражнений нужно лишь подтвердить выполнение нового подхода.

Ведение журнала тренировок
никогда еще не было таким удобным.

Хватит таскать макулатуру на тренировку!
  • У вас больше не будет необходимости вспоминать, что вы делали неделю назад.
  • Программа автоматически подставляет значения из прошлой тренировки.
  • Удобный интерфейс поможет быстро освоить принципы работы.
  • Отдельный журнал для записи биометрии.
  • Графики.
  • Возможность ставить комментарий к каждому подходу.
  • Резервное копирование.
  • Смена тем.
  • И многое другое...

Несмотря на внешнюю простоту, приложение предоставляет большую степень свободы. Если вам не хватает упражнений из готовой базы, Вы можете с легкостью добавить свое, вызвав меню и выбрав пункт "Добавить упражнение". Забыли добавить упражнение перед началом тренировки? Перейдите в список упражнений и добавьте его прямо во время занятий.

Если у Вас возникли какие-либо трудности в использовании приложения или есть предложения по его улучшению, пишите мне на электронную почту с пометкой "Training Note".
